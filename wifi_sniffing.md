# Wifi Sniffing

## Wifi Sniffing with Android

### GNURoot Debian
#### setup
```shell
$ apt-get update
$ apt-get install python
$ apt-get install python3-pip
$ apt-get install python3-requests
$ apt-get install python-requests
$ apt-get install git
$ apt-get install aircrack-ng
$ airmon-ng start wlan0
$ airodump-ng mon0
```
